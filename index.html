Project on Console

To set up Visual Studio, you have to first download it. Then you make a folder and open that folder in Visual Studio using File->Open folder. Then you add index.html or whatever else you need. Then, start coding!

The console page is a section fo the screen which displays everything you wrote in your code using the console function. You can access it on a website by right clicking, then selecting Inspect, and then at the top, click Console.
